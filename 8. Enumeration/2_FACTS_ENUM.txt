

FACT 1: TWO OR MORE ENUM NAMES CAN HAVE SAME VALUE (NOT SAME NAME)

FACT 2: WE CAN ASSIGN VALUES IN ANY ORDER. ALL UNASSIGNED NAMES WILL GET VALUE AS VALUE OF PREVIOUS NAME + 1

FACT 3: ONLY INTEGRAL VALUES ARE ALLOWED.

FACT 4: ALL ENUM CONSTANTS MUST BE UNIQUE IN THEIR SCOPE. REDECLARATION OF A NAME WITHIN SAME SCOPE IS NOT ALLOWED.
